# 7 Plugins und Tools

Plug-ins sind kleine Zusatzprogramme, die die Funktionen von
Webanwendungen erweitern. Installiert man ein Plug-in, erhält die
jeweilige Website oder Software in der Regel eine neue Funktion, die sie
vorher nicht hatte. Ein Vorteil von Plug-ins ist, dass Sie Programme und

das Plug-in ausgelagert -- eine Deinstallation der Erweiterung stellt
